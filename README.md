# clibs

Collections of useful C data structures.

All the libraries will be sorted by licenses. Here is a list of items.

* AVL Tree (CDDL)
* AVL Tree (BSD, **partly**)
* 2-3 Tree (BSD, **partly**)
* Double-Linked List (CDDL)

If marked with **partly**, it means only part of the functions are
implemented.
